# Учебный конфигурационный язык (вариант №28)

## 1. Общее описание
Инструмент командной строки на Python, который читает текст на учебном конфигурационном языке из `stdin`, преобразует его в JSON и выводит в `stdout`. При лексических или синтаксических ошибках выводит сообщение в `stderr` и завершает работу с кодом 1.

## 2. Описание функций и настроек
- Поддерживаемые конструкции языка:
  - Многострочные комментарии: `{{! ... }}`.
  - Числа: `0` или восьмеричные `0[o0][0-7]+`.
  - Имена (идентификаторы): `[a-z][a-z0-9_]*`.
  - Массивы: `list( значение, значение, ... )` (допускается пустой список `list()`).
  - Словари:
    ```
    begin
      имя := значение;
      ...
    end
    ```
  - Объявления констант: `имя = значение`. Программа состоит из набора таких объявлений; имена становятся ключами итогового JSON.
  - В выражениях можно ссылаться на ранее объявленные константы по имени.
- Основные файлы:
  - `config_lang.py` — лексер, парсер и вычисление констант.
  - `main.py` — CLI для запуска преобразования.
- Зависимости: используется только стандартная библиотека Python 3.10+.

## 3. Команды сборки и запуска тестов
- Запуск конвертации (чтение из `stdin`, вывод в `stdout`):
  ```bash
  python main.py < input.conf > output.json
  ```
- Запуск тестов:
  ```bash
  python -m unittest
  ```

## 4. Примеры использования
- Пример конфигурации веб-сервера: `examples/web_server.conf`
- Пример конфигурации IoT-устройства: `examples/iot_device.conf`
- Где можно применять:
  - Учебные кейсы по формальным языкам и парсерам.
  - Простейшие декларативные конфигурации для сервисов или устройств.
  - Демонстрация трансляции в JSON для интеграции в пайплайны.
- Быстрый пример inline:
  ```text
  x = 0o10
  y = list(0, x)
  cfg = begin
    size := x;
    items := y;
  end
  ```
  Вывод:
  ```json
  {
    "x": 8,
    "y": [0, 8],
    "cfg": {
      "size": 8,
      "items": [0, 8]
    }
  }
  ```
